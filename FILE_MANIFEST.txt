【核心常数与硬编码值】
================================================================================

API Base URL:
  https://relay.daomessage.com

WebSocket URL:
  wss://relay.daomessage.com/ws?uuid={uuid}&token={token}

密钥派生路径：
  Ed25519: m/44'/0'/0'/0/0  （身份认证）
  X25519:  m/44'/1'/0'/0/0  （消息加密）

SLIP-0010 HMAC 密钥：
  "ed25519 seed"

会话密钥推导参数：
  salt = SHA256(conversationId)
  info = "securechat-session-v1"
  length = 32

BIP-39 参数：
  熵大小：16 字节（128 bit）
  词库大小：2048 个英文单词
  PBKDF2 轮数：2048
  Passphrase：无（空字符串）

WebSocket 心跳：
  间隔：30 秒
  帧类型：ping

重连策略：
  最大尝试次数：10
  退避公式：baseDelay * 2^(count-1)
  baseDelay：1 秒

消息回执自动发送：
  delivered - 接收方客户端收到后
  read - 应用层调用 markAsRead()
